# Qadmin后台模板 - Spring Web MVC示例

这里使用的 Spring 2.4.4  thymeleaf 模板引擎作为示例		

官网[www.qadmin.net](http://www.qadmin.net)

## 安装

```shell script
mvn install
```

## 编译

```shell script
mvn package
```

## 运行

```shell script
java  -jar target/demo-0.0.1-SNAPSHOT.jar
```

## 演示

```shell script
http://127.0.0.1:8080/
```
